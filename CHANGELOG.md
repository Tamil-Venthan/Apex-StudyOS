# Changelog

All notable changes to Apex StudyOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-18

### Added

- Advanced Pomodoro timer with customizable durations
- Immersive fullscreen focus mode
- Keyboard shortcuts for timer control (Space, R, S, F, M)
- Subject tracking with modules and topics
- Analytics dashboard with daily and weekly breakdowns
- Smart workflow features and quick notes
- Glassmorphism UI design with dark mode
- Framer Motion animations
- SQLite database with Prisma ORM
- Video progress tracking for topics
- Important topics flagging system
- Rich text notes with TipTap editor
- Progress visualization with circular progress bars
- PDF export functionality for reports

### Changed

- Upgraded to modern tech stack (React 19, Electron 39, TypeScript 5)
- Improved sidebar navigation with collapsible feature
- Enhanced UI/UX with smooth animations

### Fixed

- Database initialization issues in production builds
- Timer state persistence across sessions
- Subject list update behavior

## [1.0.0] - Initial Release

### Added

- Basic study tracking functionality
- Subject and topic management
- Simple timer implementation
- Local data storage
